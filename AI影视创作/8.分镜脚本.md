### 第一部分：为什么要做分镜？（误解澄清 + 三大价值）
- **常见误解**：分镜不是只有好莱坞才需要的东西，也不是要你画得像样——几个方框、线条、箭头就够。
- **三大价值**（已在上面前言中详细展开）：
  - 避免“生成完发现不对劲、全部重来”的悲剧。
  - 每个镜头有明确任务，提示词写作瞬间变快。
  - 把“模糊感觉”翻译成AI能精确执行的画面描述。

### 第二部分：分镜四段式结构——所有好故事的骨架
无论2小时电影还是60秒短视频，好故事都遵循**建立 → 发展 → 高潮 → 结尾**（人类大脑处理故事的自然节奏）。作者给出每个段落的**核心任务、需要完成的工作、镜头特点、AI提示词方向、常见时长占比**，超级实用。

1. **建立（Setup）**  
   - **核心任务**：这是谁？在哪里？有什么问题正在酝酿？  
   - **工作**：介绍人物、交代时间地点世界观、埋下冲突种子（刚好够用，留下空白让观众想继续看）。  
   - **镜头特点**：多用远景/全景建立空间感，中景介绍人物，节奏平稳，为后续积蓄能量。光线色调建立整片基调。  
   - **AI提示词方向**：大远景，深夜城市俯拍，万家灯火，冷蓝色调，宁静而孤独，电影感，慢速。  
   - **时长占比**：20～25%。

2. **发展（Development）**  
   - **核心任务**：冲突升级，情绪积累，让观众跟着人物“着急”。  
   - **工作**：展示人物行动选择、不断加码情绪、展示性格、制造至少一次转折。  
   - **镜头特点**：景别丰富（中景+特写交替），运镜更有设计感，节奏加快，光线张力增强。  
   - **实用技巧**：用“阶梯式升级”——每隔几个镜头就把压力提高一个档次，避免“太平”。  
   - **AI提示词方向**：中景，人物快步穿过走廊，手中紧握文件，侧光强调紧张感，节奏加快，电影感。  
   - **时长占比**：40～50%。

3. **高潮（Climax）**  
   - **核心任务**：情绪能量最高点，所有张力在这里爆发/释放。  
   - **工作**：核心冲突决定性时刻、人物关键选择/真相、情绪强度峰值。  
   - **镜头特点**：特写/大特写密集，剪辑节奏最快（动作）或最慢（情感），光线/色调最戏剧性。  
   - **关键认知**：高潮不一定是“最热闹”，而是“情绪密度最高”。铺垫越充分，爆发力越强。  
   - **AI提示词方向**：极致特写，角色眼中有泪光，强侧光，深色背景，情绪在面部流动，慢速，电影感。  
   - **时长占比**：15～20%。

4. **结尾（Resolution）**  
   - **核心任务**：情绪落点，给观众“出口”和最后印象。  
   - **工作**：情绪释放/余韵、暗示变化、留下思考空间。  
   - **镜头特点**：节奏放慢，从特写拉回中/远景（抽离感），光线色调与建立呼应（或对比）。  
   - **技巧**：不需要解释一切，留白更有力量。  
   - **AI提示词方向**：缓慢拉镜头，从人物特写拉回全景，夕阳余晖，暖色调，宁静，叙事余韵，电影感。  
   - **时长占比**：10～15%。

### 第四部分：从脚本到提示词——完整实操案例（超级干货）
故事创意：一个年轻人在深夜收到一封陌生人的信，打开后陷入沉思。  
**Step 1**：确定情绪弧线——平静 → 好奇 → 困惑与震动 → 若有所思。  
**Step 2**：拆分6个关键镜头（完整走四段式）。作者给出每个镜头的**叙事任务 + 景别 + 运镜 + 光线色调 + 完整AI提示词**（可直接复制）：

1. **建立**：大远景，深夜城市俯拍，万家灯火中一扇窗户亮着，冷蓝色调，宁静而孤独，电影感，静止镜头。  
2. **建立**：中景，年轻男性坐在书桌前，台灯暖光，周围堆满书，背景暗，专注低头的姿态，静止镜头，电影感。  
3. **发展**：中景，男子缓缓注意到桌上一封白色信封，停顿，拿起翻转查看，暖色台灯侧光，轻微推镜头，悬念感，电影感。  
4. **发展**：大特写，手指慢慢撕开信封边缘，暖光，手部有微微颤抖，慢速，悬念感，电影感。  
5. **高潮**：特写，男子面部，眼神从平静到震惊到若有所思，侧光明暗各半，光影在面部流动，无台词，纯情绪特写，极缓慢推镜头，电影感。  
6. **结尾**：缓慢拉镜头，从人物特写拉回全景，他独自坐在灯下一动不动，信纸在手中，窗外夜色深沉，冷暖交界光，静默感，电影感。

**核心工作流顺序**（必须遵守）：先想故事 → 定情绪弧线 → 拆镜头序列 → 最后写提示词。很多人把顺序搞反了，导致视频只是“好看镜头堆砌”。

### ❌ 新手常见的分镜错误（6大避坑指南）
1. 跳过分镜直接生成（最致命，90%重来悲剧可避免）。  
2. 每个镜头只想“好看”，忘了“任务”（先问叙事任务，再问怎么好看）。  
3. 高潮出现得太早（铺垫不够，后续没得可用）。  
4. 发展段落太平（流水账，无情绪阶梯式升级）。  
5. 忘记结尾（观众感觉“故事突然断了”）。  
6. 分镜做好了，提示词漏掉光线（AI随机破坏情绪基调）。
